# Cake Coven Services Website

A static HTML/CSS/JS website for Cake Coven Services.

## Folder Structure

- `index.html` - Homepage
- `services.html` - Services page
- `pricing.html` - Pricing page
- `about.html` - About Ashley page
- `rewards.html` - Hexley's Rewards page
- `gallery.html` - Gallery placeholder
- `faq.html` - FAQ & Policies page
- `quote.html` - Quote request page
- `css/` - Organized stylesheets
- `js/` - Navigation script
- `assets/` - Logos, Ashley photo, Hexley art, gallery assets

## Deployment

Upload all files and folders to the existing GitHub repository root.
Cloudflare should redeploy automatically after commit/sync.
